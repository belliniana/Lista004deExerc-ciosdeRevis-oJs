# Lista de Exercícios - Arrow Functions (JavaScript ES6+)

Este projeto contém exercícios básicos em JavaScript com foco em Arrow Functions e lógica fundamental.

---

## Objetivo

Praticar a sintaxe moderna do JavaScript utilizando Arrow Functions, incluindo:

- Conversão de funções tradicionais para Arrow Functions
- Retorno implícito
- Uso de parâmetros simples e múltiplos
- Operadores condicionais (ternário)
- Template strings
- Manipulação básica de valores

---

## Importância para o aprendizado

Esses conceitos são base para o desenvolvimento moderno em JavaScript e são amplamente utilizados em:

- React
- Manipulação de dados em APIs
- Funções de callback
- Métodos de array como `map`, `filter`, `reduce`

---

## Estrutura dos Exercícios

Os exercícios podem ser organizados em um único arquivo:


exercicios.js


---

## Conteúdo dos Exercícios

### 1. Dobro
Conversão de uma função tradicional para Arrow Function.

### 2. Saudação fixa
Função sem parâmetros que retorna uma string fixa.

### 3. Soma simples
Arrow Function com dois parâmetros.

### 4. Verificar par
Função que retorna `true` ou `false` com base em uma condição.

### 5. Quadrado
Uso de retorno implícito.

### 6. Maioridade
Uso de operador ternário dentro da Arrow Function.

### 7. Conversão de moeda
Multiplicação simples com valor fixo.

### 8. Template string
Função que retorna uma frase formatada.

### 9. Multiplicação por 10
Função de uma única linha.

### 10. Comprimento de string
Retorno do tamanho de uma string.
